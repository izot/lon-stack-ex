Readme - LonTalkStack Simple Device Example using a native LonTalk Interface 

DESCRIPTION:	
 The SimpleLtDevice example implements a simple LonTalkStack device using a 
 native Layer 2 LON interface accessed using OpenLDV. The application software 
 and LON Interface Developer project for this example are shared with the 
 SimpleIp852Device example and can be found in Examples\SimpleDevice.  See also 
 Examples\SimpleDevice\readme.txt
